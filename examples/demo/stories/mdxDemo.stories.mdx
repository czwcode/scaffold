给子组件提供上下文，必须是 RdxView 的父级组件。

###  Props

  - initializeState?: { [key: string]: any }
    设置初始化的的值到Provider中。
  - value?: { [key: string]: any }
    受控的方式，设置值到Provider中，当值改变的的时候，当前context的所有组件将被重新执行。
  - onChange?: (state: {[key: string]: any}) => void
    当内部任务流状态全部执行结束的时候，onChange将被调用。
